Cara Penyelesaian Spltv dari Iklan Produk


Persamaan


  1. x + y + z= 110.000
  2. x + y + z=145.000
  3. x + y + z=110.000

Penyelesaian:

  1. Karena paket 1 dan paket 3 sama, maka kita hanya gunakan: (1) x + y + z = 110 (2) x + y + z = 145
  2. Kurangi persamaan (2) – (1): (x + y + z) – (x + y + z) = 145 – 110 → z = 35k
  3. Sekarang masukkan nilai z ke persamaan (1): x + y + 35 = 110 → x + y = 75k

Harga rak pajangan (z) = 35k Gabungan harga lampu bunga + hiasan jamur = 75k Harga x dan y tidak bisa ditentukan satu per satu, karena tidak ada persamaan tambahan yang berbeda. Agar harga x dan y bisa diketahui, diperlukan 1 persamaan lagi (misalnya paket lain atau harga salah satu produk)
