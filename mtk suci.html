Cara Penyelesaian Spltv dari Iklan Produk


Persamaan


  1. x + y + z= 155.000
  2. 2x + y + 2z=205.000
  3. x + y + 2z=220.000

Penyelesaian:

  1. Eliminasi y dan z dari persamaan pertama dan kedua: Kurangkan persamaan pertama dari persamaan kedua: (2x + y + z) - (x + y + z) = 205 - 155 x = 50
  2. Substitusi nilai x ke persamaan pertama dan ketiga: Persamaan pertama: 50 + y + z = 1 155 y + z = 105 Persamaan ketiga: 50 + y + 2z = 220 y + 2z = 170
  3. Eliminasi y dari persamaan yang dihasilkan: Kurangkan persamaan (y + z = 105) dari persamaan (y + 2z = 170) : (y + 2z) - (y + z) = 170 - 105z = 65
  4. Substitusi nilai z ke persamaan y + z = 105: y = 40 y + 65 = 105

Harga cermin (x) adalah 50. Harga tas (y) adalah 40. Harga hiasan kupu-kupu (z) adalah 65
